*para adicionar arquivos no cmd do git é bem simples, basta apenas digitar dentro das "" + extenção que deseja, como por exemplo: "MeuCodigo.txt"

*para visulizar os arquivos se estão ou nao comitados dentro do repositorio, basta apenas digitar o comando= git status
